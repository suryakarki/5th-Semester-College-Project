#About the program:

The program identifies noun corresponding to the given ampersanded pronouns in between the N line paragraph which is read from the given input file. The number N is given at the start of the input file.

#Sample Input:
4
&my& name is ram goyal. &my& best friend name is shyam. &i& read in geu. &he& reads in gehu.
ram,shyam,geu,gehu

#Output:
ram
ram
ram
shyam

#Programing Language used:

C
